# Variable assignment
- Variables are created first time they're used
- A Variable declaration doesn't include a type
- Types are inferred

`
foo = 43
bar = "this is a nice string"
`
# Comments
Comments are created with a `#`

`#This is a comment and it lasts until end of the line`
